# AI Quality Daily — Companion Code

This is the code that goes alongside the [AI Quality Daily](https://rajeshkartha.substack.com) newsletter.

## Projects

- **[careertailer/](./careertailer/)** — the resume-tailoring app. The original project, used as a reference throughout the RAG Evaluation series and as a counter-example in the AI Agents series.
- **[jobscout/](./jobscout/)** — the job-search agent built across the AI Agents series. Grows from a single LangGraph agent (v1) to a full multi-agent system with MCP, A2A, observability, and production hardening (v7).

Each project is independently runnable. See its README for setup.

## About the newsletter

AI Quality Daily covers practical guides on evaluating, testing, and shipping AI systems. Two active series:

- **RAG Evaluation** — six dimensions of RAG quality, building test datasets, scorecards, and fixing broken retrieval
- **AI Agents** — building, evaluating, and shipping agentic systems
